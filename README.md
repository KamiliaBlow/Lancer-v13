У меня есть ведро говёной мотивации, платная иишка и желание играть на сраной новой версии программы без необходимости поиска модулей для Foundry V12!. Скорее всего будут баги, немало. Если решили этим пользоваться, сообщайте об ошибках, постараюсь разобраться.
