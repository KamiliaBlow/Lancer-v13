# Lancer-v13
Так как релиз официальной версии Lancer RPG для Foundry V13 выйдет хрен пойми когда (а скорее всего никогда, ибо вышла уже версия 14). Я решила взять дело в свои руки. У меня есть ведро говёной мотивации, платная иишка и желание играть на сраной новой версии программы без необходимости поиска модулей для Foundry V12!. Скорее всего будут баги, немало. Если решили этим пользоваться, сообщайте об ошибках, постараюсь разобраться.
