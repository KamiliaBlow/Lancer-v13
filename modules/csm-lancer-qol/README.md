## LANCER QoL

This module is a collection of automation and visual effects that can help you run a great LANCER game in Foundry VTT.
Wreck automation, token special effects, and status automations are just a few of the features it can add to your game.

## [LANCER QoL Wiki](https://gitlab.com/csmcfarland/csm-lancer-qol/-/wikis/home)

The wiki page has an overview of this module, setting descriptions and any other instructions on how to use it.
